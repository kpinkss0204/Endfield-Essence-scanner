## 🛠 엔드필드 자동 스캔 시스템 가이드

이 프로그램은 게임 내 아이템의 옵션을 OCR(광학 문자 인식)로 읽어, 설정된 데이터베이스와 일치할 경우 자동으로 **잠금(Lock)** 처리를 수행하는 도구입니다.

### 1. 사전 준비 (필수)

프로그램 실행 파일과 같은 폴더에 아래 2개의 이미지 파일이 반드시 있어야 합니다.

* **`lock_template.png`**: 아이템 슬롯에 표시되는 작은 '잠금 아이콘' 이미지.
* **`lock_button_template.png`**: 옵션창 우측 하단의 '잠금/해제 버튼' 이미지.

### 2. 사용 방법

1. **테서랙트 설치**: `C:\Program Files\Tesseract-OCR` 경로에 Tesseract가 설치되어 있어야 합니다.
2. **게임 세팅**: 인벤토리를 열고, **아이템 하나를 클릭**하여 우측에 옵션 상세 창이 나오게 합니다.
3. **프로그램 실행**: GUI 창이 뜨면 템플릿 로드 여부를 확인합니다.
4. **스캔 시작**: 게임 화면을 띄운 상태에서 **`F1`** 키를 누릅니다.
* 자동으로 노란색 등급 아이템과 옵션 영역을 감지하여 스캔을 시작합니다.


5. **중단**: 스캔 중 **`F1`** 또는 **`F2`**를 누르면 즉시 중단됩니다.

### 3. 주요 기능

* **DPI 대응**: 윈도우 디스플레이 배율에 상관없이 정확한 좌표를 클릭합니다.
* **자동 그리드 감지**: 노란색(전설 등급) 바를 추적하여 아이템 위치를 스스로 파악합니다.
* **고급 전처리**: OCR 인식률을 높이기 위해 이미지를 3배 확대 및 이진화 처리합니다.
* **오타 보정**: `atribute`, `criticai` 등 OCR 오류를 자동으로 교정하여 매칭합니다.

### 4. 단축키 요약

| 단축키 | 기능 |
| --- | --- |
| **`F1`** | 자동 스캔 시작 / 일시 정지 |
| **`F2`** | 스캔 강제 종료 및 UI 초기화 |

---

**주의사항**: OCR 특성상 배경이나 글씨 겹침에 따라 인식률이 달라질 수 있습니다. 반드시 처음 몇 개가 제대로 잠기는지 확인 후 사용하세요.
