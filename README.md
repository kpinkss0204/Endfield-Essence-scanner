# 🛡️ Endfield Scanner Pro

**게임 내 아이템 옵션을 실시간으로 스캔하여 최적의 무기를 매칭하는 도구입니다.**

## ✨ 핵심 기능

* **실시간 OCR**: Tesseract를 이용해 화면 속 영문 옵션을 한국어로 자동 번역.
* **고급 이미지 처리**: OpenCV로 노이즈를 제거하여 인식률 극대화.
* **자동 매칭**: 감지된 옵션 조합을 DB와 대조하여 일치하는 무기 표시.
* **편의성**: 단축키(`F1` 시작/영역지정, `F2` 중지) 지원.

## 🚀 사용법

> [!IMPORTANT]
> **준비 사항:** 스캔 정확도를 위해 **게임 내 언어 설정을 반드시 'English(영어)'로 변경**해 주세요.

1. **F1** 누르기 → 분석할 옵션 영역을 마우스로 드래그.
2. 실시간으로 **감지된 옵션**과 **일치하는 무기** 확인.
3. **F2**를 눌러 스캔 종료.

## 🛠 필수 설정

* **Tesseract 설치**: `C:\Program Files\Tesseract-OCR` 경로 확인.
* **라이브러리**: `pip install pytesseract pillow pynput numpy opencv-python`
